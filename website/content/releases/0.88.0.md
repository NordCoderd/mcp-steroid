## What's New in v0.88.0

### NPX CLI Mode

MCP Steroid can now be launched via `npx` for quick setup without manual plugin installation. The NPX bridge includes stdio proxy support, progress streaming, update checks, and telemetry — making it easier than ever to get started with MCP Steroid.

### GoLand & WebStorm Support

GoLand and WebStorm are now officially supported JetBrains products, with dedicated integration test infrastructure to ensure compatibility.

### Improved Dialog Killer

The automatic modal dialog closer has been hardened with better exception handling and EDT threading fixes. Integration tests have been expanded to cover more dialog scenarios.

### Debugger Improvements

- Fixed the "Collecting data..." race condition in debugger expression evaluation — values now display reliably after step-over
- Added MCP resources for debugger expression evaluation, step-over, and run configurations
- Added polling fallback for complex object presentation after step-over

### OCR & Screenshot Fixes

- Fixed OCR native library loading with macOS dylib alias support
- Fixed screenshot filename collisions with unique subdirectories

### Kotlin Agent Output Filter

Replaced Python-based filter scripts with a JAR-based approach for processing Claude, Codex, and Gemini stream output — improving reliability and removing the Python dependency.

### Stability & Bug Fixes

- Fixed critical resource leaks in coroutine lifecycle management
- Fixed MCP server test restart on same port
- Fixed NPX stdio framing and reduced initialization overhead
- Fixed Docker reaper multi-endpoint connectivity for builder containers
- Fixed plugin unzip to overwrite existing files, enabling test run directory reuse

### Infrastructure & Testing

- EAP smoke test for plugin compatibility validation
- Centralized plugin versions and downgraded Kotlin to match IDE bundled version
- Video streaming dashboard with branded UI and playback controls
- Refactored test infrastructure with extracted LayoutManager, GitDriver, and ConsoleDriver
- Three new debugger integration tests with improved evidence validation
- Reduced console update latency from 1s to 100ms
