  • Refactor safely: rename a symbol across 50 files in one operation, extract methods, move classes

  • Run inspections: catch real errors before committing, not just syntax issues
  • Execute tests: run and analyze test results without leaving the agent flow
  • See the IDE: screenshot capture, UI interaction, modal dialog handling
  • Create new skills: build custom agent skills with IntelliJ API examples — use more of your IDE or custom plugins, no plugin development needed

Create Powerful IntelliJ-Driven Skills

Create custom skills for your agent in minutes. Describe what you want, give it an IntelliJ API example, and let the agent iterate. No plugin development needed.

// Example: find all TODO comments in the project
val todoItems = readAction {
    val searchHelper = PsiSearchHelper.getInstance(project)
    val result = mutableListOf<String>()
    searchHelper.processCommentsContainingIdentifier("TODO") { comment ->
        result.add("${comment.containingFile.virtualFile.path}: ${comment.text.trim()}")
        true
    }
    result
}
todoItems.forEach { println(it) }
